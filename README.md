## 👋 Hey there!

I'm a **Software Engineering student at AASTMT** (GPA 3.4 / 4.0 · Class of 2027) who ships real products — not just homework assignments.

I co-founded **[Lume Egy](https://lumeegy.com)**, a live e-commerce fashion brand where I own the **entire technical stack**: storefront dev, paid-media pixel infrastructure, and fulfillment integrations. We hit **EGP 49,000+ in gross sales** and a **2.4% peak conversion rate** within months of launch.

Beyond commerce, I architect multiplayer game systems, build cross-platform mobile apps, develop full-stack web platforms, and explore systems programming — because good engineers don't stop at one layer.
